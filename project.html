Inventory System


For my first mini-project, I developed an Inventory Management System that can help small stores efficiently track and manage their stock. The goal was to create a user-friendly application that would allow users to easily add, remove, and update inventory items, keeping track of the quantity and description of each product. This mini project gave me practical experience in managing a complete application, from the design phase to implementation, and helped me better understand the principles of programming, database integration and user interface design.
